# UMMF 0.1.0-preview.1

This is the first testable foundation release of Universal Media Mod Framework.

## Included

- Windows x64 self-contained preview CLI (`ummf.exe`)
- UMMF mod manifest validator
- deterministic texture, audio and subtitle identity demonstration
- update-resilient asset matching demonstration
- JSON Schema and an example Turkish media mod
- source archive and SHA-256 checksums

## Not included yet

This preview is not a BepInEx plugin and does not modify a Unity game yet. Texture extraction/replacement, subtitle hooks and voice playback will arrive in later previews.

## What to test

1. Download `UMMF-v0.1.0-preview.1-win-x64.zip` and extract it.
2. Open PowerShell in the extracted folder.
3. Run `./ummf.exe info`.
4. Run `./ummf.exe validate ./samples/ExampleMediaMod/mod.json` and confirm it prints `VALID`.
5. Run `./ummf.exe identity-demo` and confirm it prints a stable subtitle ID.
6. Run `./ummf.exe match-demo` and confirm it reports a compatible candidate despite the simulated content-hash change.
7. Edit a copy of `mod.json`, remove its `id`, and confirm validation reports an error.

Please report the command, output and Windows version when something fails.
