# UMMF

Universal Media Mod Framework is an experimental, update-resilient modding framework for Unity games.

The project focuses on three media surfaces:

- textures and sprites
- audio clips and subtitle-triggered voice playback
- subtitles and localization text

Development starts with Unity Mono games through BepInEx 5. IL2CPP and middleware backends such as FMOD and Wwise are planned as adapters after the core contracts stabilize.

> Status: architecture bootstrap. No public release is available yet.
