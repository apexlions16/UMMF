# UMMF 0.3.0 Önizleme 1

Bu sürüm, çok sürümlü Unity çalışma zamanı host altyapısının ilk işlevsel teslimatıdır.

## Eklenenler

- Unity oyun klasörü algılama
- Unity sürümü algılama
- Mono ve IL2CPP betik arka ucu ayrımı
- x86, x64, ARM32 ve ARM64 mimari algılama
- BepInEx kurulumu ve sürüm nesli algılama
- TextMeshPro, Unity UI, Addressables, FMOD ve Wwise izleri
- `ICalismaZamaniHostu` ortak sözleşmesi
- host yetenek bayrakları
- eski BepInEx Mono host profili
- BepInEx 5 Mono host profili
- BepInEx 6 Mono host profili
- BepInEx 6 IL2CPP host profili
- yükleyicisiz Mono ve IL2CPP kurulum adayları
- `oyun-tara` komutu
- `host-demo` komutu
- Mono ve IL2CPP tarama testleri
- dört farklı host seçim testi

## Deneme

Windows paketini ayrı bir klasöre çıkarın.

```powershell
./ummf.exe bilgi
./ummf.exe host-demo
./ummf.exe oyun-tara "D:\Oyunlar\BirUnityOyunu"
```

### Beklenen `host-demo` sonucu

Dört satırda şu host aileleri seçilmelidir:

- Eski BepInEx Mono uyumluluk hostu
- BepInEx 5 Mono hostu
- BepInEx 6 Mono hostu
- BepInEx 6 IL2CPP hostu

### `oyun-tara` ile kontrol edilecekler

- Unity sürümü doğru veya `Bilinmiyor` olarak gösterilmeli.
- Mono/IL2CPP ayrımı oyunla eşleşmeli.
- Mimari doğru görünmeli.
- Kurulu BepInEx ve medya eklentileri listelenmeli.
- Yükleyici yoksa uygulama çökmemeli; kurulum adayı ve `Başlatılabilir: Hayır` sonucu göstermeli.

## Sınırlamalar

- Bu sürüm gerçek BepInEx plugin DLL'si değildir.
- Oyun dosyalarında değişiklik yapmaz.
- Doku, ses ve altyazı değiştirme henüz etkin değildir.
- BepInEx sürümü dosya sürüm bilgisinden okunamıyorsa nesil `Bilinmiyor` kalabilir.
- Her Unity sürümü test edilmiş kabul edilmez.

## Sonraki aşama

Gerçek BepInEx 5 Mono plugin hostu, harici UMMF mod klasörü, başlangıç logları ve JSON uyumluluk raporu ortak host sözleşmesine bağlanacaktır.
