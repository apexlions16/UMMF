# UMMF 0.4.0 Önizleme 1

Bu sürüm UMMF'nin ilk gerçek oyun içi çalışma zamanı hostunu sunar. Windows EXE, desteklenen BepInEx 5 Unity Mono oyununa uygun plugin DLL'sini kurar ve oyun açıldığında UMMF başlangıç/log/rapor altyapısı çalışır.

## Yeni özellikler

- BepInEx 5 Unity Mono için gerçek plugin girişi
- eski Unity oyunları için `net35` plugin
- daha yeni Unity oyunları için `netstandard2.0` plugin
- tek EXE içinde iki plugin hedefi
- otomatik hedef seçimi
- güvenli `kur`, `durum`, `rapor` ve `kaldir` komutları
- Türkçe BepInEx logları
- harici `BepInEx/UMMF/modlar` klasörü
- oyun içi `uyumluluk-raporu.json`
- kurulum öncesi `kurulum-teshis-raporu.json`
- plugin SHA-256 denetimi
- IL2CPP ve yanlış BepInEx nesline kurulum engeli

## İndirilecek dosya

```text
UMMF-v0.4.0-onizleme.1-windows-x64.exe
```

EXE kendi .NET çalışma zamanını ve gerekli UMMF plugin DLL'lerini içerir.

## İlk test

```powershell
./UMMF-v0.4.0-onizleme.1-windows-x64.exe bilgi
./UMMF-v0.4.0-onizleme.1-windows-x64.exe oyun-tara "D:\Oyunlar\OrnekOyun"
./UMMF-v0.4.0-onizleme.1-windows-x64.exe kur "D:\Oyunlar\OrnekOyun"
./UMMF-v0.4.0-onizleme.1-windows-x64.exe durum "D:\Oyunlar\OrnekOyun"
```

Ardından oyunu bir kez başlatıp kapat ve şu dosyaları kontrol et:

- `BepInEx/LogOutput.log`
- `BepInEx/UMMF/raporlar/uyumluluk-raporu.json`

Beklenen log satırları:

```text
UMMF 0.4.0-onizleme.1 başlatılıyor.
Çalışma zamanı hostu: BepInEx 5 Unity Mono
UMMF mod klasörü hazır
UMMF başlangıcı başarıyla tamamlandı.
```

## Sınırlar

- Bu sürüm yalnızca BepInEx 5 Unity Mono plugin hostunu etkinleştirir.
- IL2CPP oyun desteği host seçimi düzeyindedir; gerçek IL2CPP plugin daha sonraki sürümde gelecektir.
- Doku, ses ve altyazı değiştirme henüz etkin değildir.
- Çevrimiçi/çok oyunculu veya hile korumalı oyunlarda kullanılmamalıdır.
