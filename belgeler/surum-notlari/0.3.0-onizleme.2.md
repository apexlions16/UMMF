# UMMF 0.3.0 Önizleme 2

Bu sürüm, uygulamanın kullanıcıya doğrudan çalıştırılabilir Windows EXE olarak ulaştırılmasını düzeltir.

## İndirilecek dosya

`UMMF-v0.3.0-onizleme.2-windows-x64.exe`

Dosya tek başına çalışır. ZIP açma, kaynak kodu derleme veya ayrıca .NET kurma işlemi gerekmez.

## Yayın güvenliği

GitHub Release oluşturulmadan önce:

- çözüm derlenir,
- bütün birim testleri çalıştırılır,
- tek dosyalı Windows x64 EXE üretilir,
- EXE Windows runner üzerinde gerçekten açılır,
- `bilgi` komutunun `0.3.0-onizleme.2` sürümünü verdiği doğrulanır,
- Release içinde doğrudan `.exe` dosyasının bulunduğu denetlenir.

Bu adımlardan biri başarısız olursa Release yayımlanmaz.

## Deneme komutları

```powershell
./UMMF-v0.3.0-onizleme.2-windows-x64.exe bilgi
./UMMF-v0.3.0-onizleme.2-windows-x64.exe host-demo
./UMMF-v0.3.0-onizleme.2-windows-x64.exe oyun-tara "D:\Oyunlar\OyunAdi"
```

`oyun-tara` komutu oyun dosyalarını değiştirmez. Unity sürümü, Mono/IL2CPP arka ucu, işlemci mimarisi, BepInEx kurulumu ve seçilen çalışma zamanı hostunu raporlar.

## Henüz bulunmayan özellikler

Bu sürüm henüz oyuna yüklenen BepInEx plugin DLL'si değildir. Doku, ses ve altyazı değiştirme işlemleri sonraki sürümlerde eklenecektir.
